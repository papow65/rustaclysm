# Rustaclysm

A reimplementation of [Cataclysm: Dark Days Ahead](https://cataclysmdda.org/) using [Rust](https://www.rust-lang.org/) and [Bevy](https://bevyengine.org/)

## Goals

1. Data-compatibility with Cataclysm: DDA
2. User-friendly UI
3. Learning Rust and Bevy
